# AeyeScan LP デザイン仕様書

## ⚠️ 重要：デザインファイルについて

**このLPはデザインファイル（Figma/XD/Sketch）が存在しません。**
全てCursor上でAIを使って作成したため、デザインファイルからの画像書き出しはできません。

**コーディング方法：**
- VercelのLP（https://aeyescan-lp.vercel.app/）を参考にコーディングしてください
- 画像は下記のURLから直接ダウンロードできます
- 画像サイズはブラウザの開発者ツールで確認できます

## 画像の取得方法

### 方法1: ブラウザから直接ダウンロード

1. **VercelのLPを開く**: https://aeyescan-lp.vercel.app/
2. **画像を右クリック → 「画像を新しいタブで開く」**
3. **新しいタブで画像URLが表示されるので、右クリック → 「名前を付けて画像を保存」**
4. **適切なフォルダに保存**

### 方法2: URLから直接ダウンロード

下記のURLリストから直接画像をダウンロードできます。
ブラウザのアドレスバーにURLを貼り付けてEnterキーを押すと、画像が表示されます。
その後、右クリック → 「名前を付けて画像を保存」で保存できます。

### 画像サイズの確認方法

1. **VercelのLPを開く**: https://aeyescan-lp.vercel.app/
2. **F12キーで開発者ツールを開く**
3. **Elementsタブで画像要素を選択**
4. **Computedタブで実際のサイズ（width/height）を確認**

または、画像をダウンロード後、画像ファイルのプロパティでサイズを確認できます。
